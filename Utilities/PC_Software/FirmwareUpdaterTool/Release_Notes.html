Purpose

Tool description:
  • FWUPG is a Java based graphical tool allowing the user to update the firmware of Teseo LIV3F on the X-NUCLEO-GNSS1A1 expansion board.
  • FWUPG is platform independent and supports Windows, Mac OS X and Linux.
  • Usage: see readme
