Purpose

The GNSS1A1 BSP (Board Specific Package) drivers are parts of the X-CUBE package and provide a set of high level APIs relative to the hardware components and features in the X-NUCLEO-GNSS1A1 expansion board.
The BSP drivers allow a quick access to the board’s services using high level APIs and without any specific configuration as the link with the HAL and the external components is done in intrinsic within the drivers.
From project settings points of view, user has only to add the necessary driver’s files in the workspace and call the needed functions from examples.
