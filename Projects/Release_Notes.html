Purpose

The X-CUBE-GNSS1 Projects provide examples and applications to aid understanding how to develop an application using the GNSS Drivers and Middlewares for the Teseo-LIV3F/Teseo-VIC3DA chip. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
