Purpose

The X-CUBE-GNSS1 is an expansion software package for STM32Cube. X-CUBE-GNSS1 software features:
  • Complete middleware to build applications using either the Teseo-LIV3 or the Teseo-VIC3DA device for positioning
  • Easy portability across different MCU families thanks to STM32Cube
  • Sample applications to read data received by either the Teseo-LIV3 device mounted on the X-NUCLEO-GNSS1A1 expansion board or the Teseo-VIC3DA mounted on the X-NUCLEO-GNSS2A1 expansion board
  • Free user-friendly license terms
  • Example implementation available for the X-NUCLEO-GNSS1A1/X-NUCLEO-GNSS2A1 expansion board plugged on top of one NUCLEO-F401RE, NUCLEO-L476RG, NUCLEO-L073RZ or NUCLEO-U575ZI-Q

Update History

Main Changes

Product update

  • Remove the deprecated reference to the backup time and date source web server.
  • The A_GetPos Application for B-L475E-IOT01A1 board includes a RTC initialization mechanism which relies on the usage of an external web server. The availability of this server is not guaranteed over time. That example server name #defined in TIME_SOURCE_HTTP_POST shall therfore not be used in production.
